# Vk2.0
Задачу сам недавно реализовывал для заказчика на фрилансе. По описанию задача не большая, у тебя есть 2500 пабликов в ВК(позже дам айдишники), твоя цель, получить из всех пабликов комментарии к каждому посту, не раньше 24 часов. Т.е. текущая дата минус 24 часа. В итоге на выходе должно быть, дата комментария, сколько лайков, сам текст, и ссылка на пост. Реализовать надо так что бы отрабатывало менее чем за 30 минут.
